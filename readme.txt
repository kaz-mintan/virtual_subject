仮想被験者を作って推定結果を調べるという話です。

- v_subject.py

  - 因子F（因子じゃなくて、ゲームの状況）から、表情を出力する。
  - 各因子毎の気まぐれ度pを与えて、確率pでランダムに表情を出力してください。
    - pは、センサの精度と言い換えられます。

  - 重みセットwを与えて、任意の仮想被験者を作ってください。
    - （ある要素のwの値を0にすれば、想定因子の個数を増やしたことになります。用意した基底関数の個数に応じた推定結果の精度の変化を調べられます。たぶん）

- dummy_maker.py

  - 仮想被験者用のテストデータを作るため、ダミーの因子時系列データを作ります。
  - ダミーのM時系列も作っちゃっていいか。

- face_check.py

  - 表情推定結果と実際の表情を比較する。表情推定結果は、データ数Nを変えて推定した重みw＿Nと、virtual_factor.csvに基づき推定する。
  - もういっこキテカンの入力Mをvirtual_mental.csvにするか、estimated_phi_N.csvにするかは、どっちが良いかを考えてくれ。（重み調整時に参考にしていたのは、estimated_phi_N.csvだといえる。）

- mental_check.py

  - 気分推定結果を一望する。virtual_mental.csvは単調増加であり、それに対しどう変化したか見てくれたまえ。
